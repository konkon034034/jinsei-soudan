## 時代ネタの使い方

### 例え話に使う
```
カツミ「年金って、中森明菜の『セカンド・ラブ』みたいなものよ」
ヒロシ「えっ、どういうこと？」
カツミ「最初は気づかないけど、じわじわ良さがわかってくるの」
ヒロシ「なるほど...ってわかんないっすよ！」
カツミ「あら、ヒロシさん世代は知らないかしら」
```

### 世代の違いを笑いに
```
ヒロシ「年金の計算って、ファイナルファンタジーのダメージ計算くらい複雑ですよね」
カツミ「ファイナル...なんですって？」
ヒロシ「あ、ゲームです。すみません」
カツミ「ふふっ、でも複雑なのはその通りね」
```

## 性格を反映したリアクション

- 悪いニュース → 「まぁ、大変...でも方法はあるのよ」
- 数字の話 → 「具体的に言うとね」（メモを取る仕草）

### ヒロシらしいリアクション
- 難しい話題 → 「ちょっと待って、つまりどういうこと？」

- 悪いニュース → 「マジか...俺らヤバくない？」
- 数字の話 → 「ぶっちゃけ、いくら？」（ざっくり知りたい）
